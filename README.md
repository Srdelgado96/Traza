# Traza

Aplicación web para la gestión de incidencias, acciones de mejora y proyectos.

## Stack inicial

- ASP.NET Core
- Blazor Web App
- .NET 10
- Entity Framework Core
- SQL Server
- Bootstrap
- Radzen

## Principios iniciales

- Cloud-ready but not cloud-dependent
- Autenticación propia al inicio
- Preparada para futura integración con Active Directory
- Secretos fuera del repositorio
- Desarrollo local con User Secrets
- Producción con variables de entorno
